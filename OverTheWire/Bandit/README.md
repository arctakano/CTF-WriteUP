## Bandit

Bandit sangat friendly untuk pemula. Permainan ini akan mengajarkan dasar-dasar yang sangat dibutuhkan utnuk dapat memainkan tahapan yang lebih menantang.

# Note

Permainan ini sama seperti CTF ataupun permainan yang serupa. Anda mulai dari level 0 dan mencoba untuk memecahkan setiap masalah pada setiap level tersebut. **Perlu diinget** setiap level memiliki **Hint** untuk dapat melanjutkan kelevel selanjutnya

Anda akan menghadapi berbagai situasi dimana anda tidak tau apa yang harus anda lakukan. **KEEP CALM AND CHILL!!** karena permainan ini menajarkan anda mulai dari dasarnya. Mulai dari mempelajari bagaimana cara menggunakan tools, command, dan informasi baru. kalau anda masih baru dan belum pernah menggunakan linux sebelumnya saya sarankan untuk membacanya dengan baik seperti [bagaimana cara menggunakan linux?](https://manpages.ubuntu.com/manpages/noble/man1/intro.1.html)

Sebelum itu, ada beberapa hal yang dapat anda coba jika merasa kurang yakin, seperti:
 - Pertama, jika anda mengetahui suatu perintah tetapi merasa kurang tau cara penggunaannya? Coba untuk melihat [Manual](https://en.wikipedia.org/wiki/Man_page) (Manual Page) dengan mengetik pada command seperti `man [Perintah]`. Misalnya, `man ls` melihat dokumentasi cara menggunakan `ls`. tekan `q` untuk keluar (anda dapat menggunakan `/` untuk mencari perintah yang kurang dimengerti).
 - Kedua, jika tidak dapat menggunakan `man`, anda juga dapat menggunakan perintah `help [Perintah]`. Misalnya `help ls` ataupun  `help`.
 - Selain itu, anda juga dapat menggunakan [Google](https://www.google.com/) untuk mencari dan mempelajari cara menggunakannya
