## Level 0

**Level Goal:** Tujuan dari level ini adalah agar anda masuk ke dalam permainan menggunakan SSH. Anda hanya perlu menghubungkan `bandit.labs.overthewire.org` pada port `2220`. Username `bandit0`
dan kata sandinya `bandit0`. setelah itu, buka halaman [level 0 > 1](https://overthewire.org/wargames/bandit/bandit1.html)

- Host : bandit.labs.overthewire.org
- Port : 2220
- Username : bandit0
- password : bandit 0

# The Prosess

Lakukan koneksi remote dengan menggunakan Protokol [SSH](https://ubuntu.com/server/docs/openssh-server)

```bash
$ ssh bandit0@bandit.labs.overthewire.org -p 2220
```
